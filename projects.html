Here are some of the projects that I've made so far.

libft

A function library project in C. Includes 50+ basic C-functions, such as atoi, itoa, memcpy etc.

get_next_line

A school project to create a C-function that reads a file line by line and returns the strings one by one.

fillit

A program that can place up to 27 tetris blocks into the smallest square possible.

printf

In this project I recreated the functionality of the 'printf' function in C. Includes most of the conversions.

filler

A program that plays a game against other programs via a VM. The goal is to place tetris-like blocks on an arena, the program that places the most blocks wins.

lem_in

A program that finds the optimal path to move ants through a maze of rooms. Uses a variation of different pathfinding algorithms.

corewar

A two part project that includes a compiler for Assemply-like programs and a virtual machine that runs the compiled programs in a game.

roger_skyline

A school project where I set up a server on a virtual machine, manage it, protect it and host a website on it.

this website

I originally made this website for a school project that required me to host a website on my own server. I since then expanded this to what it is now.
